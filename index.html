      Connect your Kaggle account

      1. Sign in at kaggle.com (free account).
      2. Open Account → Settings → API and create a new token.
      3. Save the file as ~/.kaggle/kaggle.json
      4. Or paste the token into ~/.kaggle/access_token

      Run chmod 600 on the credentials file if downloads fail.
